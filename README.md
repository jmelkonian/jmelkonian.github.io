This repository contains a simple academic profile template suitable for an economics job-market page.

Files added:

- [index.html](index.html) — main profile page template
- [css/styles.css](css/styles.css) — stylesheet
- [images/profile.svg](images/profile.svg) — placeholder image
 - [research.html](research.html) — full research experience and projects
 - [teaching.html](teaching.html) — teaching experience and courses
 - `melkonian_resume_late_may2026.pdf` — uploaded CV (linked from the site)

Customize the page by editing `index.html` (name, contact, research, links) and replacing the SVG at `images/profile.svg` with your portrait (JPG/PNG) of the same filename.

Preview locally:

```bash
# from repository root
python3 -m http.server 8000
# open http://localhost:8000/ in your browser
```
